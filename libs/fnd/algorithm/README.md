# Bksge.Algorithm

このライブラリはstd::algorithmの独自実装またはラッパーです。

ほとんどの場面でstd::algorithmと同じように使えます。

### std::algorithmからの変更点

* 可能な限りconstexprをつけた
* C++の特定のバージョン以降で追加された関数をC++のバージョンに関係なく使えるようにした

* 一部の関数の引数を変更

* 関数を追加
