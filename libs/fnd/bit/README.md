# Bksge.Bit

このライブラリはstd::bitの独自実装またはラッパーです。
C++20で追加されたstd::bitを、C++11以降であれば使えるようにします。

### 依存ライブラリ

* Bksge.Config
* Bksge.CStdDef
* Bksge.CStdInt
* Bksge.CString
* Bksge.TypeTraits
